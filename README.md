## Configuration ğŸ› ï¸
Keybindings:
**The Leader key is spacebar key**

- Ctrl + s: Save changes.
- Ctrl + n: Toggle file explorer.
- Ctrl + p: Open fuzzy file finder.
- ctrl + w: Toogle between editor and filesystem
- Tab : Switch between the open files in bufferline
